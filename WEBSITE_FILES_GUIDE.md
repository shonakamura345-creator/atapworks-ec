## 📁 主要なファイルと役割

### トップページ（メインページ）

**ファイル**: `app/page.tsx`
- **役割**: トップページの表示を制御
- **内容**: 
  - HOMEセクション（「出版イベント開催決定」の部分）
  - 本の紹介セクション
  - SCHEDULEセクション
  - GOODSセクション（商品一覧）
  - GAMEセクション（ミニゲーム）
  - CONTACTセクション

**変更方法**: このファイルを編集すると、トップページの内容が変わります。

---

### ヘッダー（ナビゲーション）

**ファイル**: `app/components/Header.tsx`
- **役割**: ページ上部のナビゲーションバー
- **内容**: 
  - ロゴ（「Sho建築士」）
  - メニュー（HOME, SCHEDULE, GOODS, CONTACT）
  - カートアイコン

**変更方法**: このファイルを編集すると、ヘッダーの内容が変わります。

---

### フッター

**ファイル**: `app/components/Footer.tsx`
- **役割**: ページ下部のフッター
- **内容**: 著作権表示など

**変更方法**: このファイルを編集すると、フッターの内容が変わります。

---

### ミニゲーム

**ファイル**: `app/components/RunningGame.tsx`
- **役割**: 「Sho建築士 街を走るゲーム」のミニゲーム
- **内容**: ゲームのロジック、描画、UI

**変更方法**: このファイルを編集すると、ミニゲームの内容が変わります。

---

### 商品データ

**ファイル**: `app/data/products.ts`
- **役割**: 販売する商品のデータ
- **内容**: 商品名、価格、説明、画像パスなど

**変更方法**: このファイルを編集すると、商品一覧が変わります。

---

### 商品詳細ページ

**ファイル**: `app/store/[id]/page.tsx`
- **役割**: 個別商品の詳細ページ
- **URL**: `https://atapworks.co.jp/store/[商品ID]`

**変更方法**: このファイルを編集すると、商品詳細ページの表示が変わります。

---

### カートページ

**ファイル**: `app/cart/page.tsx`
- **役割**: ショッピングカートページ
- **URL**: `https://atapworks.co.jp/cart`

**変更方法**: このファイルを編集すると、カートページの表示が変わります。

---

### チェックアウトページ

**ファイル**: `app/checkout/page.tsx`
- **役割**: 決済情報入力ページ
- **URL**: `https://atapworks.co.jp/checkout`

**変更方法**: このファイルを編集すると、チェックアウトページの表示が変わります。

---

### レイアウト（全ページ共通）

**ファイル**: `app/layout.tsx`
- **役割**: 全ページ共通のレイアウト
- **内容**: 
  - HTMLの基本構造
  - ヘッダーとフッターの読み込み
  - Google Analyticsの設定

**変更方法**: このファイルを編集すると、全ページの共通部分が変わります。

---

### スタイル（CSS）

**ファイル**: `app/globals.css`
- **役割**: 全ページ共通のスタイル
- **内容**: Tailwind CSSの設定やカスタムスタイル

**変更方法**: このファイルを編集すると、全ページのスタイルが変わります。

---

## 🎨 よく変更するファイル

### トップページの内容を変更する場合
→ `app/page.tsx` を編集

### ヘッダーのメニューを変更する場合
→ `app/components/Header.tsx` を編集

### 商品を追加・削除・変更する場合
→ `app/data/products.ts` を編集

### ミニゲームを変更する場合
→ `app/components/RunningGame.tsx` を編集

### 全体的なデザインを変更する場合
→ `app/globals.css` を編集

---

## 📝 変更の流れ

1. **ファイルを編集**
   - Cursor（IDE）でファイルを開いて編集

2. **ローカルで確認**
   ```bash
   npm run dev
   ```
   - ブラウザで `http://localhost:3000` にアクセス
   - 変更が反映されているか確認

3. **コミット・プッシュ**
   ```bash
   git add .
   git commit -m "変更内容の説明"
   git push origin main
   ```

4. **本番環境に反映**
   - Vercelが自動的にデプロイ
   - 数分後に `https://atapworks.co.jp` で確認
